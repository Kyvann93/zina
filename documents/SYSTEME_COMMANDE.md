## 💻 Comment Utiliser

### 1. Accéder à la Page de Commande
```
http://localhost:5000/commander
```

### 2. Se Connecter
- Scanner le QR code du badge OU
- Entrer le matricule, nom et département

### 3. Parcourir le Menu
- Sélectionner une catégorie
- Utiliser les filtres
- Rechercher un plat spécifique

### 4. Ajouter au Panier
- Cliquer sur le bouton "+"
- Ajuster les quantités
- Consulter le total

### 5. Commander
- Cliquer sur "Commander"
- Choisir le mode de paiement
- Sélectionner l'heure de récupération
- Confirmer la commande

### 6. Récupérer la Commande
- Se rendre à la cantine
- Présenter le numéro de commande
- Récupérer le repas

## 🔧 Personnalisation

### Ajouter un Nouveau Plat
```javascript
{
    id: 66,
    name: "Nom du Plat",
    description: "Description détaillée",
    price: 5000,
    category: "lunch",  // breakfast, lunch, snacks, salads, drinks, desserts, specials
    image: "🍽️",
    available: true,
    popular: false,
    prepTime: 15
}
```

### Modifier les Prix
Dans `static/js/ordering.js`, modifier la propriété `price` de chaque item.

### Changer les Catégories
Modifier le tableau `menuDatabase` et mettre à jour les boutons de catégorie dans le HTML.

## 🐛 Dépannage

### Le panier ne se met pas à jour
```javascript
// Vérifier que updateCartUI() est appelé
console.log(cart);
updateCartUI();
```

### La commande ne passe pas
```javascript
// Vérifier la connexion API
fetch('/api/order', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(orderData)
})
.then(response => response.json())
.then(data => console.log(data));
```

### Problème de connexion
- Vérifier que le sessionStorage est activé
- Vider le cache du navigateur
- Réessayer la connexion
