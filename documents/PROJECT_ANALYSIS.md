# Zina - Premium Food Delivery Platform

## Project Overview
Zina is a modern food delivery web application built with Flask (Python backend) and vanilla JavaScript (frontend). It uses Supabase as the database backend and provides a complete ordering experience with an elegant user interface.

## Current Features & User Experience

### 🎨 **User Interface & Design**
- **Modern Landing Page**: Beautiful gradient background with animated circular menu selection
- **Responsive Design**: Mobile-friendly layout with smooth animations
- **Professional Styling**: Clean, modern UI with Font Awesome icons and custom CSS animations
- **Interactive Elements**: Hover effects, smooth transitions, and micro-interactions
- **Zina Branding**: Premium blue color scheme and elegant design language

### 🍽️ **Menu System**
- **Category-Based Navigation**: Four main categories (Breakfast, Lunch, Snacks, Three Course)
- **Visual Category Selection**: Circular menu interface with icons for intuitive navigation
- **Dynamic Menu Loading**: Real-time menu data fetching from backend API
- **Product Display**: Grid layout showing product images, names, prices, descriptions, and preparation times
- **Product Options**: Support for customizable options with additional pricing

### 🛒 **Shopping Cart**
- **Real-time Cart Management**: Add/remove items with quantity controls
- **Cart Sidebar**: Slide-out cart with item list and total calculation
- **Visual Feedback**: Cart count badge updates automatically
- **Toast Notifications**: User-friendly feedback for cart actions

### 💳 **Order Processing**
- **Order Placement**: Complete checkout process with order submission
- **Payment Integration**: Payment method selection and processing
- **Order Confirmation**: Real-time order status updates and confirmation messages
- **Order Tracking**: API endpoints for order status retrieval

### 🗄️ **Backend Architecture**
- **RESTful API**: Well-structured API endpoints for all operations
- **Database Integration**: Supabase for data persistence with proper relationships
- **Data Models**: Comprehensive data models using Python dataclasses
- **Error Handling**: Proper exception handling and error responses

### 📊 **Data Management**
- **User Management**: User profiles with authentication support
- **Product Catalog**: Full product management with categories and options
- **Order Management**: Complete order lifecycle management
- **Payment Processing**: Payment status tracking and updates

## Technical Stack

### Backend
- **Flask**: Web framework for Python
- **Supabase**: Database and backend services
- **Python-dotenv**: Environment variable management
- **Dataclasses**: Type-safe data modeling

### Frontend
- **Vanilla JavaScript**: No framework dependencies, pure JS
- **CSS3**: Modern styling with animations and transitions
- **Font Awesome**: Icon library
- **Responsive Design**: Mobile-first approach

### Database Schema
- **Users**: User profiles and authentication
- **Categories**: Product categorization
- **Products**: Menu items with pricing and availability
- **Product Options**: Customizable product variations
- **Orders**: Order management and tracking
- **Payments**: Payment processing and status
- **Order Items**: Detailed order line items

## User Experience Flow

1. **Landing Experience**: Users see an elegant landing page with circular category selection
2. **Category Selection**: Click on desired meal type (Breakfast, Lunch, Snacks, Three Course)
3. **Menu Browsing**: View products in selected category with details and pricing
4. **Cart Management**: Add items to cart with real-time updates
5. **Checkout Process**: Review cart and place order
6. **Order Confirmation**: Receive confirmation and order details
